# ディレクトリ構成 :

 main : パック関連

 level : ワールドに依存する系

 entity : 敵とプレイヤー以外のエンティティ

 player : プレイヤー全般

 enemy: 敵エンティティ全般>
  __behavior__ : 振る舞いやAIなどを"敵の名前"か"種類名"のフォルダ下に保存

 item : アイテム系 アイテムの機能など


# Tag名:

 先頭大文字のタグ(例: Entity) タグの種類 基本EntityかEnemy

 小文字.小文字(例: entity.liquid_mana) それの名前(id)


# CustomModelData:
 10000..19999 : MagicBook

 90000..99999 : その他
